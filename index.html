Sudoku Solver

This project resulted from the attempt to take the sudoku puzzle solving algorithm a step further. Rather than simply taking a 2D matrix of digits and solving the puzzle, this program is able to take an image of the sudoku puzzle (png, jpg, tiff, etc) and uses the Tesseract Optical Character Recognition API to read each cell and store the resulting digit. Once the image is processed into a 2D vector of integers, the puzzle can be solved normally. Several complications arose because of Tesseract is unable to distinguish the numbers from each other as well the lines which make up the graph. To solve this problem I utilized the OpenCV C++ library to pre-process the image by removing all all horizontal and vertical lines from the image while also processing each individual digit.
